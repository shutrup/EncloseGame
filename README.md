# Enclose

Enclose is a Dots and Boxes style strategy game focused on native iOS.

## Project Structure

- `/Users/sarap005931/Desktop/Enclose/Enclose` — SwiftUI iOS app source
- `/Users/sarap005931/Desktop/Enclose/Enclose.xcodeproj` — Xcode project
- `/Users/sarap005931/Desktop/Enclose/Enclose/Resources` — localized strings and audio assets

## Development

1. Open `/Users/sarap005931/Desktop/Enclose/Enclose.xcodeproj` in Xcode.
2. Select target `Enclose`.
3. Run on simulator/device.

Build from terminal:

```bash
xcodebuild build -scheme Enclose -project Enclose.xcodeproj -destination 'platform=iOS Simulator,name=iPhone 17 Pro'
```
